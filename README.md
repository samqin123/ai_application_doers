# AI应用实践仓库

本仓库将沉淀和记录各种AI技术方向的应用评测、产品评价推荐，以及实际案例素材，方便实践者（doers）能够快速复现和应用。

## 目录结构说明

- **AI+MED** - 医疗领域AI应用
  - AI-mdt: AI辅助医疗决策
  - Patients_toolkits: 患者工具包
  - RAG_Med: 医疗领域RAG应用
  - knowledge-base: 医疗知识库

- **AI+RAG** - 检索增强生成技术应用
  - Dify: Dify平台应用
  - Fastgpt: FastGPT平台应用
  - graphrag: 图谱RAG应用
  - nativerag: 原生RAG应用

- **AI-MKTING** - AI营销应用
  - AI+advertising: AI广告应用
  - AI-Insights: AI洞察分析
  - AI-marketing: AI营销策略
  - AIGC_contents: AIGC内容生成

- **AI_Coding_tools** - AI编程工具评测
  - 多种AI编程辅助工具的实践案例和评测

- **DEMO_CASES** - 演示案例
  - 实际应用案例的演示和说明

- **MCP&SKILLS** - 工具和技能
  - mcps: MCP工具集
  - skills: 各类技能实践

- **claude-scientific-skills** - Claude科学技能模块
  - 从外部仓库集成的Claude科学技能相关工具和资源
  - 该模块通过git子模块方式集成

## 仓库目标

1. 记录AI技术的实际应用效果
2. 提供可复现的技术案例
3. 分享产品使用体验和评测
4. 为实践者提供参考和指导

## 使用说明

每个目录下包含了对应技术方向的实践案例、评测报告和复现指南，欢迎探索和使用。
