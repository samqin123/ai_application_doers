## 这是什么黑科技？

说起来很简单：**Amp + Claude Skills front-end**

**Amp** 是什么？一个代码编辑器，能让 Claude AI 帮你写代码。

**Claude Skills front-end** 是什么？就是一份"前端设计秘籍"，告诉 Claude：毛玻璃怎么写、布局怎么排、常见坑怎么避。

**效果是啥？** 不是 Claude 变聪明了，而是**Claude 有了"工作指南"**，就像给实习生配了一个主办级别的规范文档。结果就是：一次到位，不用反复改。

### 怎么用？3 行命令

```bash
# 1. 打开你的 Amp（代码编辑器）
# 2. 导入 frontend-design Skill
amp skill add anthropics/skills/frontend-design

# 3. 把开发提示词（web_dev_prompt.md）扔给 Claude
# Claude 自动按规范生成代码
```

就这么简单。

## 技术细节（可跳过，给 geek 看）

### Flexbox：90% 的人都做错了

```javascript
/* ❌ 错误 - 导致重叠 */
.card { justify-content: space-between; }

/* ✅ 正确 - 合理分配空间 */
.card { flex-direction: column; overflow: hidden; }
.fixed { flex-shrink: 0; }
.flexible { flex: 1; }
```

### 毛玻璃效果怎么实现

```css
backdrop-filter: blur(10px);           /* 模糊背景 */
background: rgba(255, 255, 255, 0.15); /* 半透明 */
border: 1px solid rgba(255, 255, 255, 0.2);
```
